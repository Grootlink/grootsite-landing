# Grootlink Landing - LIBRE (AWS Auth Edition)

Esta es la landing page oficial de **LIBRE**, potenciada por el ecosistema **Grootlink**. 

## Características
- **Identidad AWS**: Integración conceptual y visual con AWS Rekognition para verificación biométrica.
- **Design System**: Estética premium con modo oscuro, glassmorphism y micro-animaciones.
- **Optimizado para Móviles**: Diseñado para lucir increíble en cualquier dispositivo.
- **Ready for Coolify**: Incluye configuración para despliegue inmediato.

## Despliegue en Coolify
Para desplegar este proyecto en Coolify:
1. Conecta este repositorio.
2. Selecciona el servicio como **Static Site** o utiliza el **Dockerfile** incluido.
3. ¡Listo!

## Tecnologías
- HTML5 / CSS3 (Vanilla)
- Google Fonts (Syne, DM Sans)
- AWS Rekognition Branding
